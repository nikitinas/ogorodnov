Структура сайта

1. ГЛАВНАЯ СТРАНИЦА

Герой-баннер: Крупное, вдохновляющее видео, где попеременно показаны все ЦА (дирижирующий педагог, поющий ребенок, выступающий руководитель, сосредоточенный психолог, актер на сцене). Крупный заголовок: «Ваш голос может все. Раскройте его потенциал».

Навигация по сегментам («Выберите свой путь»): 7 интерактивных карточек-иконок, ведущих в основные разделы. При наведении карточка подсвечивается, и появляется краткое описание («Для родителей», «Для музыкантов» и т.д.).

О методике за 90 секунд: Короткий анимированный ролик, объясняющий суть трех китов методики (Алгоритм, Тактирование, Жесты) через инфографику и живые кадры.

Ключевые преимущества (в цифрах/фактах): «+95% детей начинают петь чисто», «+80% руководителей отмечают рост уверенности в переговорах», «В 3 раза снижается утомляемость голоса у педагогов».

Истории успеха: Карусель с 3-4 самыми сильными и разными историями (ребенок, руководитель, актер).

Призыв к действию (омнибус): «Не знаете, с чего начать? Пройдите бесплатный 5-минутный тест и получите персональный план развития вашего голоса».

2. РАЗДЕЛЫ ДЛЯ КАЖДОЙ ЦЕЛЕВОЙ АУДИТОРИИ
   Каждая страница раздела имеет идентичную структуру, но уникальное визуальное и текстовое наполнение.

2.1. Подзаголовок и ключевой месседж (например, для родителей: «Раскройте природный слух вашего ребенка»).

2.2. Блок «Ваша проблема — наше решение»: Список с иконками (Проблема: «Ребенок фальшивит» -> Решение: «Чистое интонирование через ладо-вокальные жесты»).

2.3. Блок «Как это работает именно для вас»: Адаптированное объяснение трех ключевых упражнений с примерами.

2.4. Истории успеха сегмента: Фильтруемая галерея с текстовыми, видео- и аудио-историями.

2.5. Форматы участия и CTA: Четкие блоки с ценами и действиями («Групповые занятия», «Индивидуальные уроки», «Онлайн-курс», «Скачать бесплатный урок»).

3. ИНТЕРАКТИВНАЯ ЛАБОРАТОРИЯ
   Центр бесплатного вовлечения и демонстрации методики.

«Алгоритм-Конструктор»: Интерактивный тренажер.

«Ритм-Мастер»: Тренажер тактирования.

«ЛадоГрад»: Тренажер жестов.

Доступ к полным версиям — по регистрации или подписке.

4. О МЕТОДИКЕ

Подробно об Д.Е. Огороднове.

Научное обоснование (цитаты из PDF, ссылки на исследования).

Глоссарий терминов («Алгоритм», «Тоника» и т.д.).

Библиотека с материалами для скачивания (статьи, схемы).

5. БЛОГ / МЕДИА

Умная лента публикаций. Статьи автоматически тегируются по ЦА. В шапке фильтр: «Показать материалы для: [Родителей] [Педагогов] и т.д.».

Вебинары и онлайн-курсы.

6. КОНТАКТЫ

Форма обратной связи с обязательным полем «Мой запрос относится к» (выпадающий список с ЦА).

Карта сертифицированных педагогов/тренеров (с фильтрацией).

Запись на первичную консультацию.

Технические и маркетинговые «фишки»
Умный чат-бот: При входе на сайт бот задает 2-3 вопроса: «Здравствуйте! Вы хотите развить голос для себя, для ребенка или для профессиональных целей?». На основе ответа он сразу направляет пользователя в нужный раздел.

Личный кабинет: Единый кабинет для всех. Здесь пользователь видит купленные курсы, прогресс в «Лаборатории», рекомендации (например, «Вам, как руководителю, будет полезен курс «Голос лидера»).

Кросс-сегментные рекомендации: В разделе для «Психологов» система может рекомендовать статью из раздела «Антистресс», так как темы пересекаются.

PWA (Progressive Web App): Чтобы сайт работал как быстрое и удобное приложение на любом устройстве.

Эта структура создает впечатление масштабной, профессиональной и научно обоснованной экосистемы, где каждый, от ребенка до CEO, чувствует, что метод создан специально для него, являясь частью большого сообщества людей, работающих над своим голосом.
